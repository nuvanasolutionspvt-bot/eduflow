Excel Format

Use only .xlsx files in one of these exact column orders:

name, father_name, dob, gender, student_class, section, roll_no, admission_no, address, mobile, status
name, father_name, dob, gender, student_class, section, roll_no, admission_no, address, mobile, photo, status

  • You can also use the downloadable full template if you want to include extra fields like mother name, religion, caste, birth place, and previous school.
  • dob format: YYYY-MM-DD (example: 2012-06-15)
  • gender: male, female, or other
  • student_class: 1 to 12
  • section: A, B, C, D, E, F
  • photo optional: absolute path, project-relative path, or media-relative path (example: students/photos/aarav.jpg)
  • status: active or inactive
  • admission_no must be unique
